C++一些特性

委托构造函数 Delegating Construstor

可在构造函数中调用构造函数

类内初始化

成员变量的初始化能够像普通变量一样直接进行赋值。

nullptr – NULL

空指针，推荐使用nullptr

NULL可以继续赋值，这会导致意外的情况出现

枚举类

enum class Number {one ,two ,three};

强类型，不能被隐式的转化为int型

类型推导 Type Infering

但是不要滥用

auto 可用于函数返回值

constexpr

告诉编译器这个表达式本质是个常数

将运行时计算提前到编译时实现性能优化

能修饰函数，甚至时递归函数

初始化表 initializer list

STL和自建类中的数据类型可以直接列表初始化值

基于范围的for循环

类似java，以及可以配合结构化绑定

Unique ptr

auto_ptr -> unique_ptr

只会不需要手动delete这个指针

lambda表达式
